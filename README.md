# Kickstarter Campaign Success Predictor

## Overview
Analysis and predictive modeling of Kickstarter campaign outcomes using historical data.

## Project Phases
1. **Data Cleaning & Exploration** (Python/pandas)
2. **SQL Analysis** (SQL querying and insights)
3. **Predictive Modeling** (Coming soon - ML model to predict campaign success)

## Dataset
Dataset provided by Kaggle (https://www.kaggle.com/datasets/kemical/kickstarter-projects/data)

## Tools & Technologies
- Python (pandas for cleaning, sklearn for modeling)
- SQL for analysis
- Power BI (for visualizations)
